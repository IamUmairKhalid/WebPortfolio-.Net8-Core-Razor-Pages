# 🌐 WebPortfolio

A modern and responsive **Personal Portfolio Website** built with **ASP.NET Core 8 Razor Pages**. It includes a public portfolio and an admin dashboard for managing website content.

---

## 🚀 Features

- Responsive Portfolio Website
- Admin Dashboard
- Authentication System
- Profile Management
- Education & Experience Management
- Skills Management
- Services Management
- Testimonials Management
- Social Links Management
- Contact Form
- Image Upload
- Email OTP Verification
- SQL Server Database

---

## 🛠️ Tech Stack

- ASP.NET Core 8
- Razor Pages
- Entity Framework Core
- SQL Server
- Bootstrap 5
- SMTP Email Service

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/WebPortfolio.git
```

### Restore Packages

```bash
dotnet restore
```

### Update Database

```bash
dotnet ef database update
```

### Run the Project

```bash
dotnet run
```
